How to Start

django-admin startproject mb_project

Start App

python manage.py startapp posts

Project vs App

A project is the entire application. A app is a submodule of the project. The project contains apps. The demo project we will be looking at only contains one app.

To run your server locally

python manage.py runserver

You will see this first time you run locally

Models

Two Step Migration

python manage.py makemigrations posts python manage.py migrate posts

Creating a Superuser

python manage.py createsuperuser

Urls

Why Do We Have 2 urls.py Files

1. The project-level urls.py file where we simply include our posts and add include on the second line

2. The app level one where can set up the individual pages.
